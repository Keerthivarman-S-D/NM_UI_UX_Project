## Project Overview
Healthy Bytes is a personalized healthy meal delivery UI/UX design project that focuses on clean user experience, custom diets, and modern visual design.  
This case study includes user research, competitive analysis, wireframes, information architecture, and a high-fidelity animated prototype.

## User Research
![Alt Text](./comp.png)
### User Groups Interviewed
- Gym enthusiasts  
- Students  
- Working professionals  
- Elderly users  

### Key Insights
- Users want affordable healthy food
- High need for customization (less oil/salt, soft texture)
- Prefer home-style meals
- Current apps deliver cold food or charge high fees

---

## Competitive Analysis
Platforms compared:
- Fresh Green Bowl  
- Elith Foods  
- Diet Kadai  
- Freshlings  
- CultFit  

Comparison metrics:
Pricing, Delivery, Subscription, Social Media presence, Contact method.
